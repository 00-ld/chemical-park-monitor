# 数据库目录说明

本目录用于存放化工园区危险气体扩散与溯源系统的数据库相关文件。

## 目录约定

```text
db/
  README.md
  schema/       # 初始化建表脚本
  migrations/   # 版本迁移脚本
  seed/         # 基础种子数据
  test_data/    # 数据库测试数据
  er/           # ER 图或表结构说明
  backups/      # 本地备份文件，禁止提交敏感生产备份
```

## 强制要求

- 数据库使用 MySQL。
- 字符集统一使用 `utf8mb4`。
- 不得在 SQL、配置文件或文档中明文写入生产数据库密码。
- 表结构变更必须同步更新迁移脚本和相关文档。
- 测试数据必须与生产敏感数据隔离，必要时进行脱敏。
