# Twin

This directory contains digital twin integration assets and service-side notes for SuperMap iPortal, SuperMap native scenes, and future Three.js work.

Rules:
- Treat the SuperMap iPortal dashboard as the current primary 3D entry.
- Keep dashboard data contracts and coordinate mapping documents here or under `docs/`.
- Do not commit exported third-party SDK bundles, generated scene caches, or large binary tiles.
- Use stable JSON contracts from the backend instead of hardcoded frontend-only data.
