# 传感器风险等级划分标准

## 一、概述

本项目采用 **4级风险等级** 对传感器进行分类管理，风险值由系统自动计算。

**核心公式 (GB 18218-2018 第4.3.2条 公式2):**

```
R = α × (β₁×q₁/Q₁ + β₂×q₂/Q₂ + ... + βₙ×qₙ/Qₙ)
```

本项目单传感器单物质场景简化为：

```
R = α × β × (q/Q) 
```

| 参数      | 名称        | 来源                   | 取值范围       |
| ------- | --------- | -------------------- | ---------- |
| **α**   | 暴露人员校正系数  | GB 18218 表5          | 0.5 ~ 2.0  |
| **β**   | 毒性气体校正系数  | GB 18218 表3/表4       | 1.0 ~ 5.0  |
| **q/Q** | 实际存在量/临界量 | GB 18218 表1 + 设施危险等级 | 0.5 ~ 10.0 |

## 五、q/Q — 实际存在量与临界量比值

### 5.1 临界量 Q (GB 18218 表1 原文)

| 物质      | 临界量 Q |
| ------- | ----- |
| H2S 硫化氢 | 5t    |
| NH3 氨   | 10t   |
| CO 一氧化碳 | 30t   |
| CH4 甲烷  | 50t   |
| C2H4 乙烯 | 50t   |
| O2 氧气   | 200t  |

### 5.2 传感器场景 q/Q 估算

传感器本身不含危化品存量数据，利用设施 `hazardLevel` (0~1) 映射有效 q/Q：

```
q/Q = 0.5 + hazardLevel × 9.5
```

| 设施危险等级           | q/Q  | 说明             |
| ---------------- | ---- | -------------- |
| hazardLevel=1.0  | 10.0 | 满量危化品 (如H2S储罐) |
| hazardLevel=0.7  | 7.15 | 高危生产区          |
| hazardLevel=0.5  | 5.25 | 中等风险区          |
| hazardLevel=0.3  | 3.35 | 低风险区           |
| hazardLevel=0.15 | 1.93 | 办公/监测区         |

## 六、完整计算示例

### 示例1: T-08 硫化氢储罐旁传感器 (TK-17L)

```
传感器参数: detectionRange = "H2S/可燃气体/CO/O2", installationHeight = 0.3m
最近设施: t08 硫化氢储罐 (hazardLevel=0.95)

β = 5      (GB 18218 表3: 硫化氢 β=5)
α = 1.0    (GB 18218 表5: 储罐区 1~29人)
q/Q = 0.5 + 0.95 × 9.5 = 9.525
位置修正 = 1.10  (h=0.3m ≤ 0.5m, 重气低位修正)

R = 1.0 × 5 × 9.525 × 1.10 = 52.39
→ 1级 重大风险 (R=52.39 ≥ 50)
```

### 示例2: T-07 液氨储罐旁传感器 (TK-13H)

```
传感器参数: detectionRange = "NH3/CO/可燃气体/O2", installationHeight = 2.2m
最近设施: t07 液氨储罐 (hazardLevel=1.00)

β = 2      (GB 18218 表3: 氨 β=2)
α = 1.0    (GB 18218 表5: 储罐区 1~29人)
q/Q = 0.5 + 1.00 × 9.5 = 10.0
位置修正 = 1.05  (h=2.2m ≥ 2.0m, 轻气高位修正)

R = 1.0 × 2 × 10.0 × 1.05 = 21.00
→ 2级 较大风险 (10 ≤ 21.00 < 50)
```

### 示例3: 裂解车间释放源旁传感器 (PA-01L)

```
传感器参数: detectionRange = "CO/可燃气体/H2S/O2", installationHeight = 0.5m
最近设施: b06 裂解车间 (hazardLevel=0.70, personnel=35)

β = 5      (GB 18218 表3: 含H2S, 取最高β=5)
α = 1.2    (GB 18218 表5: 30~49人)
q/Q = 0.5 + 0.70 × 9.5 = 7.15
位置修正 = 1.10  (h=0.5m ≤ 0.5m, 重气低位修正)

R = 1.2 × 5 × 7.15 × 1.10 = 47.19
→ 2级 较大风险 (10 ≤ 47.19 < 50)
```

### 示例4: 综合办公楼南侧传感器 (A-01L)

```
传感器参数: detectionRange = "CO/可燃气体/H2S/O2", installationHeight = 0.5m
最近设施: b01 综合办公楼 (hazardLevel=0.15, personnel=180)

β = 5      (GB 18218 表3: 含H2S, 取最高β=5)
α = 1.5    (GB 18218 表5: 50~99人 → 1.5; 办公楼人员180但厂外暴露按区域估算)
q/Q = 0.5 + 0.15 × 9.5 = 1.925
位置修正 = 1.10  (h=0.5m ≤ 0.5m, 重气低位修正)

R = 1.5 × 5 × 1.925 × 1.10 = 15.88
→ 2级 较大风险 (10 ≤ 15.88 < 50)
```

### 示例5: 锅炉房CH4传感器 (UT-01H)

```
传感器参数: detectionRange = "CH4", installationHeight = 2.2m
最近设施: b14 锅炉房 (hazardLevel=0.30)

β = 1.5    (GB 18218 表4 W2: 易燃气体)
α = 0.5    (GB 18218 表5: 0人)
q/Q = 0.5 + 0.30 × 9.5 = 3.35
位置修正 = 1.05  (h=2.2m ≥ 2.0m, 轻气高位修正)

R = 0.5 × 1.5 × 3.35 × 1.05 = 2.64
→ 4级 低风险 (R=2.64 < 5)
```

### 示例6: 氧气制备站传感器 (UT-08)

```
传感器参数: detectionRange = "O2", installationHeight = 1.5m
最近设施: b18 氧气制备站 (hazardLevel=0.25)

β = 1.0    (GB 18218 表4 W4: 氧化性气体)
α = 0.5    (GB 18218 表5: 0人)
q/Q = 0.5 + 0.25 × 9.5 = 2.875
位置修正 = 1.0   (h=1.5m, 通用高度)

R = 0.5 × 1.0 × 2.875 × 1.0 = 1.44
→ 4级 低风险 (R=1.44 < 5)
```
